# İş Akışı

Üzerinde çalışılacak task belirlendikten sonra işletilecek akış sırasıyla aşağıdaki gibidir.

1. `05-jira.md` kuralları doğrultusunda taskı kendine assign et.

2. `05-jira.md` kuralları doğrultusunda taskı `ANALYSE` statüsüne al.

3. `03-understanding.md` kuralları doğrultusunda taskı anla.

4. `06-git.md` kuralları doğrultusunda clone/pull ve feature branch oluşturulur.

5. `04-analyse.md` kuralları doğrultusunda analiz yapılır.

6. `05-jira.md` kuralları doğrultusunda yapılan geliştirmeyi özetleyen bir yorum Jira task'ına girilir.

7. `05-jira.md` kuralları doğrultusunda task `ANALYSE DONE` statüsüne alınır.

8. `05-jira.md` kuralları doğrultusunda task unassign edilir.
