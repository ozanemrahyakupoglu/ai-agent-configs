# Rule: Development Done

## Koşullar

Task aşağıdaki koşulların tamamı sağlandığında **DEVELOPMENT DONE** statüsüne alınır:

- Geliştirme tamamlandı
- Değişiklikler `agent` branch'ine merge edildi ve push'landı
- Jira task'ına yorum girildi

## Statü Geçişi

Task **DEVELOPMENT DONE** statüsüne alınır ve assignee kaldırılır (unassign).
